----------------------------------------------------------------------------------------------------------------------------------
Emplacement			Description													
----------------------------------------------------------------------------------------------------------------------------------
/data				Fichiers de données (configurable)
/doc				Documentation
/lib				Librairies nécessaires au programme
/logs				Fichiers de LOG (et sauvegardes bases)
/JaMuz.db		    Votre nouvelle Base de données musicale (sauvegardez régulièrement !)
/JaMuz.jar		    Le programme en lui-même
/JaMuz.properties	Fichier de configuration
/JaMuz.sh		    Script de lancement pour Linux
/myMovieDb.db		Base de données pour onglet Vidéo

/JaMuz.xml			Fichier de configuration (optionnel)
						pour changer emplacement et nom de la 
						base de données. Voir modèle dans /doc.
